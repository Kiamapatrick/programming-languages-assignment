# List Implementations

This repository contains implementations of **array-based lists** and **linked lists** across multiple programming languages. Each implementation includes the fundamental operations: **insert**, **delete**, **update** and **traverse**.

## Languages Included

- **C**
- **Java**
- **Python**
- **Pascal**

## Repository Structure

- **C**
  - `array_list.c`
  - `linked_list.c`

- **Java**
  - `ArrayListImpl.java`
  - `LinkedListImpl.java`

- **Pascal**
  - `array_list.pas`
  - `linked_list.pas`

- **Python**
  - `array_list.py`
  - `linked_list.py`
